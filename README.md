# Lyft-Clone
A clone of the Lyft website made during our first week of React, utilizing mapping, props, and styling in jsx.


## Getting Started

* clone the repo
* npm install
* npm run start
* will be served to localhost 8080

### Prerequisites

```
A good code editor - I use Atom
```
```
Node Package Manager
```
